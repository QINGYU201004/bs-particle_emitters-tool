# bs-particle_emitters-tool

Набор инструментов командной строки для обработки JSON-файлов `particle_emitters`.

## Возможности

- Исправление свободного/нестандартного формата JSON с сохранением отсортированного и отформатированного вывода.
- Конвертация нового формата эмиттеров частиц в старый совместимый формат.
- Поиск и извлечение эффектов по ключу эмиттера и ключевым словам `movieclip` (`swf` / `name`).
- Извлечение уникальных SC-имен из `movieclip.swf` с экспортом в CSV-файл с заголовком.
- Безопасное слияние JSON-файлов с пропуском дублирующихся ключей.
- Многоязычное меню: английский, китайский, русский.

## Требования

- Python 3.8+

## Быстрый старт

```bash
python3 main.py
```

После запуска выберите действие в меню:

- `1`: Исправить JSON (`fix.py`)
- `2`: Конвертировать новый формат в старый (`old.py`)
- `3`: Поиск/извлечение эффектов (`search.py`)
- `4`: Физическое слияние JSON (`add.py`)
- `5`: Сменить язык
- `6`: Извлечь SC-имена из `movieclip.swf` (`sc_extract.py`)

## Примечания

- Имена выходных файлов можно задавать вручную в CLI.
- Для больших JSON-файлов время обработки зависит от производительности устройства.
